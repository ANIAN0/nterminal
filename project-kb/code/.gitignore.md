# .gitignore

## 功能与职责

忽略依赖、构建产物、运行数据、日志和历史研发工作区；精确放行 `workplace/1.3`，使当前需求、方案、任务和验收证据可追踪。

## 修改注意事项

放行子目录时必须同时放行父目录；不得放行 `data/`、`logs/`、`.agents/` 或历史 workplace。修改后使用 `git check-ignore -v` 检查正反样本。
